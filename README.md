# Password Strength Checker 🔐

## Overview
This project analyzes the strength of user passwords based on:
- Length
- Character diversity (uppercase, lowercase, digits, symbols)
- Entropy calculation

It provides recommendations to improve weak passwords.

## Features
- Detects weak, medium, and strong passwords
- Suggests improvements for better security
- Simple C implementation for learning cybersecurity basics
